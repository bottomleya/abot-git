Sick Fits

Sick Fits is a fictional online store built with ReactJS and GraphQL. The full stack application is built with a NodeJS backend alongside a GraphQL Yoga server, which handles all of my custom server side logic, such as credit card transactions via Stripe, password reset, permissions management and more. The backend also interfaces with Prisma, which is a powerful ORM library that sits on top of the database. GraphQL has been a buzz lately and I was keen to give it a try. It's essentially an alternative to the way in which REST APIs work. In contrast to hitting an API endpoint, where you just accept the data that's returned to you, GraphQL allows you to write queries on the client side and it gives you the freedom to specify the exact data in the shape that you want returned from the server. I used Jest for testing the key React components and GraphQL queries. Please refer to my GitHub repo for more information about my approach and the technologies used.

Forkify

Forkify is a vanilla JavaScript application that interacts with the food2fork API to fetch and display recipe food data. This app uses modern JavaScript tools, such as Webpack to bundle the modules, and Babel to convert ES6, ES7 and ES8 back to ES5. The user can search for a specific recipe, and add ingredients to a shopping list or save to a favourites list via local storage. I created a simple Express server in order to host the app via Heroku. Please refer to my GitHub repo for more information about my additional and re-factored features.
